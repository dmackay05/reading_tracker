Setup Instructions

1. Go to script.google.com → New project.
2. Paste this code, then deploy as a Web App (execute as you, access: anyone): function doGet() { const sheet = SpreadsheetApp.openById('YOUR_SHEET_ID').getSheets()[0]; const data = sheet.getRange('A1').getValue(); return ContentService.createTextOutput(data || '{}') .setMimeType(ContentService.MimeType.JSON); } function doPost(e) { const sheet = SpreadsheetApp.openById('YOUR_SHEET_ID').getSheets()[0]; sheet.getRange('A1').setValue(e.parameter.data); return ContentService.createTextOutput('ok'); } 3. Replace YOUR_SHEET_ID with your Google Sheet ID (found in its URL).
4. Copy the deployed web app URL and paste it in the field above.
